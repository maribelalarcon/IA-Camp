# IA-Camp Web Project

A web development project using HTML, CSS, and JavaScript.

## Project Structure

```
src/
├── css/        # Stylesheets
│   └── style.css
├── js/         # JavaScript files
│   └── main.js
├── images/     # Image assets
└── index.html  # Main HTML file
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Start developing!

## Technologies Used
- HTML5
- CSS3
- JavaScript ES6+
